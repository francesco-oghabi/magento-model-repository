# Magento 2 Model Repository & API Interface

Boilerplate implementing the repository pattern in Magento 2 / Adobe Commerce:
Model, ResourceModel and Repository wired to the `Api/Data` and `Api` service
interfaces — the clean way to expose entities via service contracts.
